Smart Application Desktop

Description:
Smart Application Desktop est une application graphique développée en Python permettant la manipulation, l’analyse et la visualisation de plusieurs algorithmes de graphes.

Ce projet a été réalisé dans un cadre académique afin de renforcer la compréhension des structures de données, des algorithmes et de leur implémentation avec interface graphique.

L’application combine calcul algorithmique et visualisation interactive pour offrir un outil pédagogique clair et structuré.


Objectifs du projet:
- Implémenter les principaux algorithmes de graphes
- Créer une interface graphique intuitive avec Tkinter
- Visualiser dynamiquement les résultats
- Comprendre l’optimisation et l’analyse des réseaux
- Développer une architecture logicielle structurée


Fonctionnalités:
- Création manuelle d’un graphe
- Génération aléatoire de graphe
- Visualisation graphique interactive
- Sélection des sommets source et destination
- Calcul du taux de connexité (%)
- Export des résultats


Algorithmes implémentés:
- Kruskal (Arbre couvrant minimal)
- Dijkstra (Plus court chemin)
- Bellman-Ford
- Welsh-Powell (Coloration de graphe)
- Ford-Fulkerson (Flot maximal)
- MPM (Méthode du chemin critique)


Technologies utilisées:
- Python
- Tkinter
- NetworkX
- Matplotlib


Installation:
git clone https://github.com/EL-OGRI-Mohammed/Smart-Application-Desktop.git
cd Smart-Application-Desktop
pip install networkx matplotlib
python main.py
